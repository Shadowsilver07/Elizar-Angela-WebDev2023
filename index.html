### Github 1. VScode open 2. Create folder 3. Git bash open 4. Git init 5. Git add . 6. Git commit -m "first commit" git remote -v [to check the remote] git branch [To check the branch] 7. git remote add origin https://github.com/Shadowsilver07/AngelaBauer_Story.git - this pulls the main branch 8. git checkout -b API - this creates a new branch called API 9. git add . 10. git commit -m "API branch created" 11. git push -u origin API - this pushes the API branch to the remote repository Additional: 1. To change remote repo git remote set-url origin https://github.com/username/new-repo.git git remote -v 2. rename the remote instead of changing its URL, git remote rename old-name new-name 3. add a new remote instead of changing an existing one git remote add new-name
